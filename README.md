# edms

A minimalist electronic document management system.  The goal of this system is to allow you to be able to store your documents electronically instead of taking up space in your home.  Retrieval is easy due to basic metadata and tagging functionality.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
